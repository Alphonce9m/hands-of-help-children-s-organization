# Hands of Help

A modern website for Hands of Help, a community organization dedicated to empowering communities through education, digital literacy, and support programs.

## Features

- Responsive design for all devices
- Modern UI with Tailwind CSS
- Interactive components
- SEO-friendly
- Fast performance with Next.js

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- ESLint

## Getting Started

### Prerequisites

- Node.js 18.17 or later
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/hands-of-help.git
cd hands-of-help
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Run the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
